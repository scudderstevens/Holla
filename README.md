## Development Notes

**Vercel/Next.js**

to run locally:

`% cd holla-tyson/site`

`% yarn dev` Starts the development server.

if another webpack instance is running on port 3000, change to 8000

$ `npm run dev -- -p 8000`

**Sanity**

`% cd holla-tyson/studio`

`% sanity start` Starts the development server.
